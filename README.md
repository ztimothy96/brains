# Brains

Let's segment some brain tumors! 

## Data

Download images and segmentation files from [link](https://springernature.figshare.com/collections/A_comprehensive_dataset_of_annotated_brain_metastasis_images_with_clinical_and_radiomic_data/6194104/1) and unzip in project root folder.

For more information about the dataset, see [Nature paper](https://www.nature.com/articles/s41597-023-02123-0).
